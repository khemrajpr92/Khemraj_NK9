## Assessment 1: English–Hindi Dataset Processing and Analysis

### What it does
1. Loads the parallel `eng.txt` and `hin.txt` files (10,000 line-aligned sentence pairs).
2. Computes **word counts** for each English and Hindi sentence.
3. Keeps only sentence pairs where **both** word counts fall in the range **3–60**.
4. Computes the **word count difference** (English − Hindi) and keeps only pairs
   where the difference falls within **-10 to +10**.
5. Computes **character counts** for both languages and their difference.
6. Exports the final cleaned dataset to Excel.

### How to run
```bash
cd assessment1
pip install pandas openpyxl
python process_dataset.py
```
This produces `cleaned_english_hindi_dataset.xlsx` in the same folder.

### Results
| Stage | Rows |
|---|---|
| Raw input pairs | 10,000 |
| After word-count range filter (3–60, both languages) | 9,644 |
| After word-count difference filter (-10 to +10) | **8,976** |

### Output columns
`English Sentence`, `Hindi Sentence`, `Word Count (English)`, `Word Count (Hindi)`,
`Difference between Word Count (English) and Word Count (Hindi)`,
`Character Count (English)`, `Character Count (Hindi)`,
`Difference between Character Count (English) and Character Count (Hindi)`

## Assessment 2: Translation with LLM

### What it does
1. Samples **200 English sentences** from the Assessment 1 cleaned dataset
   (`sampled_200.xlsx`, random_state=42 for reproducibility).
2. Translates each sentence into Hindi using an **LLM (Claude, by Anthropic)**
   — Google Translate and Helsinki-NLP were **not** used, per the assessment rules.
3. Scores the translations against the original reference Hindi sentences using
   **BLEU**, **CHRF**, and **TER** (via the `sacrebleu` library).
4. Saves the final output to Excel (English | Hindi translation) and the scores
   to a `.txt` file.

### LLM used
**Claude (Anthropic)**. Translations were generated by prompting the LLM with
each English sentence individually and recording its Hindi output, stored in
`claude_translations.json`.

### How to run
```bash
cd assessment2
pip install pandas openpyxl sacrebleu
python translate_with_llm_final.py
```
This produces `translation_output.xlsx` and `translation_scores.txt`.

|---|---|
| **BLEU** | 23.7456 |
| **CHRF** | 49.8365 |
| **TER**  | 66.7081 |

*(Number of sentences evaluated: 200)*

### Output columns
`Original English sentence`, `Model-generated Hindi translation`
